# React Cheese API

- A simple API for CRUD operations on cheese data with a React frontend.
- The API is hosted on Heroku.

## Technologies Used

- Node
- Express
- Mongoose
- CORS
- Morgan
- Heroku

## Heroku URL

- [See live](https://react-cheese-api.herokuapp.com/api/cheese/)
